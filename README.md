This is blog website where user can write an posting with suitable title and uses css ,html ,javascript to modify webpages according to the need. 
It is run on https://localhost:3000 server after initiating npm node modules
https://localhost:3000/home  : the page where users content can be added
https://localhost:3000/about : the page that can used to describes about why the website has been done
https://localhost:3000/contact : this page contains developer contact details so user can contact for any queeries.
https://localhost:3000/compose : In this page user can compose postings which going to be added on the home page
